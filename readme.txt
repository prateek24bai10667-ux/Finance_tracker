# ExpenseWise: Java Personal Finance Tracker

### Project Overview
ExpenseWise is a simple Java-based command-line application that helps a person manage their daily finances by tracking income and expenses. The application solves a real-world problem by providing a simple data persistence solution for managing finances.

### Features
- Income and Expense tracking with custom categories.
- Calculates the Net Balance automatically.
- Data Persistence is achieved by Serializing data and storing it in 'finance_data.ser'.
- Validates input to prevent the application from crashing due to invalid data.

### How to Run
- Make sure JDK 11+ is installed on your machine.
- Compile the following Java files: `FinanceTracker.java` and `Transaction.java`.
- Run the application by executing `FinanceTracker.java`.
